# AI Markdown & LaTeX Copier

A Tampermonkey userscript for copying AI responses as clean Markdown, with better LaTeX formula support.

It is designed for users who often copy content from ChatGPT, Claude, DeepSeek, Qwen, Moonshot and similar AI websites into Markdown editors such as Obsidian, Typora, VS Code, or Notion.

## Installation

1. Install the Tampermonkey browser extension.

2. Create a new script in Tampermonkey.

3. Copy the full script from this repository and paste it into the Tampermonkey editor.

4. Save the script.

That’s it. Open a supported AI website, select the content you want to copy, and press `Ctrl + C`.

If it works correctly, the copied content will be converted into cleaner Markdown text, including code blocks, lists, tables, links, and LaTeX formulas.
